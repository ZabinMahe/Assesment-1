5. Marathon Energy Tracker
Bright Future School is organizing a 5-day mini marathon challenge. Each day, a student runs a certain number of kilometers. You need to compute the running sum of the distances.
The running sum means that each element in the output list is the cumulative sum of all previous elements including the current one.
Input Format (User Input)
●	The first line contains an integer n, representing the number of days.
●	The second line contains n space-separated integers representing the distance run each day.

Output Format
●	Print the running sum list.
 Example Input

6. The Hidden Pair Challenge
In a school math competition, students were given a box of numbered cards. Each card contained a hidden integer.
The challenge was simple but tricky.
The teacher announced:
“Pick two different cards such that the sum of their numbers equals the target value.
 You cannot use the same card twice, and there is exactly one correct solution.”
Instead of checking every possible pair, a smart student decided to solve the puzzle efficiently by remembering the numbers she had already seen.
She scanned the cards one by one.
For each number, she asked:
Is there another number I have seen before that can pair with this number to reach the target?
If yes, she found the answer.
If not, she stored the current number and continued searching.
Finally, she discovered the two cards that formed the target sum and won the challenge.
Sample Input 1
2 7 11 15
9
Sample Output 1
[0, 1]
